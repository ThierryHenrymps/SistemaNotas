# Sistema Notas
## Descricao
Esse projeto funciona para escolas facilitando a adicao e o gerenciamento dos alunos,
assim podendo ter um relatorio total dos alunos com nome, nota, matricula de cada um.

## IDE
Java 21, IDE IntelliJ atualizado.

## Ultilizacao
Apos a inicializacao, aparece um menu interativo ara emitir boletins de todos os alunos,
exibir quantidade de alunos aprovados, exibir aluno com a maior media,
lancar notas para um aluno pelo indice, sendo ideal para o gerenciamento
do aluno, por fim a finalizacao do projeto.

## Descricao de Classes
### TesteNotas(MAIN)
O arquivo TesteNotas e a main do projeto nele esta a estrutura principal de inicializacao presente no projeto. 
Nele temos varias chamadas de funcoes para organizacao dos alunos. Switch com varios cases e cada um com suas funcoes para a 
utilizacao total do projeto, facilitando o gereciamento dos alunos.

## Pasta Controller
Na pasta controller fica as configuracoes como aluno e gerenciamento de turma, la tem construtores, Getters and Setters,
o coracao do projeto nele fica a estrutura onde eu puxo depois para a main.

### Aluno
Nessa pasta temos contrutores defult ou adicionado pela pessoa que esta ultilizando, por Getters and Setter, 
la fica salvo dados para a criacao de alunos.

### GerenciadorTurma
Nele fica o gerenciamento dos alunos na sala, assim podendo ver se esta lotada, a adicao do aluno na sala,
e a listagem pelo arrays.

## Pasta Util
Nele contem tudo aquilo que eu possa ultilizar como menu e etc.

### Menu
Ele como o proprio nome diz e um menu interativo onde fica como static para a ultilizacao dele na main.
